2. **[効率的な開発段階](./development-stages.md)**

6. **[Claude Code Memory Tips](./recommand-claude-memory.md)**

## 🎯 このガイドの目的

- Claude Code と効果的に協業する方法の提示
- アジャイル環境での AI ツール活用戦略
- 品質の高いコード生産のためのベストプラクティス

## 🚀 始め方

1. まず[準備事項と環境設定](./preparation.md)を読んで開発環境を構成してください。
2. [効率的な開発段階](./development-stages.md)を通じて基本的なワークフローを理解してください。
3. アジャイルチームなら[アジャイル開発戦略](./agile-strategy.md)を参考にしてスプリントを計画してください。

## 💡 核心原則

1. **小さな単位から開始**: 全体システムを一度に要求せず、コンポーネント単位でアプローチ
2. **明確なコミュニケーション**: 具体的で詳細な要求事項の伝達
3. **反復的改善**: 完璧な最初の実装よりも継続的な改善に集中
4. **コンテキスト提供**: 既存コードとプロジェクト構造に関する情報共有
