## 🚀 Getting Started

1. First, read [Preparation and Environment Setup](./preparation.md) to configure your development environment.
2. Understand the basic workflow through [Efficient Development Stages](./development-stages.md).
3. If you're on an agile team, refer to [Agile Development Strategy](./agile-strategy.md) to plan your sprints.

## 💡 Core Principles

1. **Start Small**: Approach component-by-component rather than requesting the entire system at once
2. **Clear Communication**: Deliver specific and detailed requirements
3. **Iterative Improvement**: Focus on continuous improvement rather than perfect first implementation
4. **Provide Context**: Share information about existing code and project structure
